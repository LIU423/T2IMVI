# Reliability Analysis Data Directory

This directory contains reliability analysis artifacts (e.g., experiment results).

## Structure

```
reliability_analysis/
├── comparison/
│   └── <project_name>/
└── results/
    ├── experiment1/
    ├── experiment1_numerical/
    └── experiment2/
```

## Note

This directory is ignored by git (see `.gitignore`).
Keep large or generated results here and do not commit them.
