# Output Data Directory

This directory contains output data generated by the T2IMVI pipeline.

## Structure

```
output/
├── IRFL/                   # IRFL processing results
├── phase0/                 # Phase 0 outputs
└── phase1_extraction/      # Phase 1 extraction outputs
    ├── output/
    │   ├── literal/
    │   └── figurative/
    └── checkpoints/
```

## Note

This directory is ignored by git (see `.gitignore`).
Output files will be generated when running the pipeline.
