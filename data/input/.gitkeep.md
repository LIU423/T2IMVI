# Input Data Directory

This directory contains input data for the T2IMVI pipeline.

## Structure

```
input/
└── IRFL/
    ├── non_none/          # Non-null idiom data
    │   └── unique_idioms.json
    └── matched_images/    # Matched images for analysis
```

## Note

This directory is ignored by git (see `.gitignore`). 
Please obtain the IRFL dataset separately and place it here.
